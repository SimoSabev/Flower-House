03 — Технический стек и интеграции
Слой Технология Назначение Приоритет
Frontend (клиент) React + Next.js Сайт + конструктор букета + ЛК клиента Критичный
Frontend (флорист) React PWA / iPad app Планшетный POS-интерфейс, очередь заказов Критичный
Мобайл (курьер) React Native / Expo Список доставок, навигация, статусы Высокий
Backend / API Node.js + Fastify REST API, бизнес-логика, очереди задач Критичный
База данных PostgreSQL + Redis Основные данные + кэш остатков в реальном времени Критичный
Хранилище S3 / Cloudflare R2 Фото цветов, фото доставки, иллюстрации букетов Высокий
Уведомления Twilio / WhatsApp API SMS, WhatsApp Business: статусы заказов Высокий
Оплата Stripe / Revolut Business Онлайн-оплата, рефанды, POS-терминал Критичный
Аналитика Metabase / Recharts Дашборд владельца, отчёты, прогнозы Средний
Авторизация Clerk / Auth.js Роли: клиент, флорист, курьер, админ Высокий
